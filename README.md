## Key Features

* React Router Dom.
* Firebase Authentication 
  - Email/Password
  - Google Auth
* React Firebase Hooks  
* Stripe Payment Gateway
* [KaTeX](https://khan.github.io/KaTeX/) Support
* React Bootstrap
* React Toastify
* AOS Animation
